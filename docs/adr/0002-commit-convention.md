# Tag: Description commit format

Commits mirror the Daily Log line item format directly: `tag: Description`. No scope, no footer.

```
feat: Seed tabbed content block from theme
```

**Tags:** `feat`, `fix`, `bug`, `chore`, `docs`, `style`, `refactor`, `perf`, `test`, `infra`

Work items in the Daily Log provide per-ticket grouping; commits stay clean and tool-agnostic.
